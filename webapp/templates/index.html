About

This site purpose is to provide end users documentation of functions included in the K8 product from Kerridge Commercial Systems.

Anyone who has used Kerridge extensively knows that there is little documentation of functions that can be used in the Report Generator and DX applications.

These functions can be extremely powerful and useful. By extracting these functions this website presents them in a more human readable way, allowing users to also search for specific keywords.

Users can also add examples to existing functions or submit useful code snippets that they have created.
